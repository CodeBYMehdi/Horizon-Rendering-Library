HRL Engine

High-quality multi-API rendering engine for professional C/C++ applications. Designed to be minimal, flexible, and future-proof.

Note

This project was developed by a single passionate developer. I’ve tried to make everything work smoothly, but there may still be bugs.

If you encounter any issues or have suggestions, feel free to contact me at: oscarsoirey.contact@gmail.com

Version

HRL is currently in pre-alpha version 0.1. Only the OpenGL backend is available at the moment.
